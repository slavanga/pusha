Pusha

Pusha is a lightweight but complete off-canvas solution.

Download | View on GitHub

Highlights

  • Smooth transitions
  • No dependencies
  • Left, right, top and bottom positioning
  • Four modes: push, pull, overlay and fade
  • Custom styling using Sass variables and mixins
  • Support for fixed elements
  • iOS rubber band effect disabled by default

Browser Support

  • Chrome
  • Firefox
  • IE 9+
  • Edge
  • Safari (Mac)
  • Android 4+
  • iOS 8+
